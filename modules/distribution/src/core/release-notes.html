Release Note - Entgra IoT Server Version 3.7.0

We are pleased to announce Entgra IoT Server 3.7.0.

Entgra IoT Server is the successor of WSO2 IoT server and includes capability to manage mobile devices(MDM), manage applications(MAM) and IoT devices in a single environment. It provides best of breed technologies for device manufacturers to develop connected smart products as well as anyone looking for a well established EMM solution to manage devices in their organisations. Entgra IoTs 3.7.0 pays special focus to Kiosk devices and many other EMM improvements.

What's new in Entgra IoTS 3.7.0

  • New operation to add Android/iOS application configuration
  • Ability to turn off or on backing up data for Android
  • Automate Android enrollment by using QR code
  • New policy to install fonts for iOS
  • Ability to control voice roaming on iOS
  • Ability to control Data roaming on iOS
  • Ability to control wifi hotspot on iOS
  • Ability to control bluetooth on iOS
  • Ability to change iOS wallpaper
  • Ability to install MacOs Applications
  • New policy to add iOS network usage rules
  • Active Sync integration for iOS
  • New policy to restrict the app store of mac-os device
  • New Restrictions for Windows Devices
  • Disable Adding NonMicrosoft Accounts Manually
  • Disable Private Browsing
  • Disable Removable Drive Indexing
  • Restricts the users from changing language settings
  • Disables region settings on a device
  • Disables Cortana from the windows search bar
  • Ability to create managed preferences on mac os device
  • Ability to shut down MacOs device
  • Ability to restart MacOs device
